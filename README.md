# Custom Drink Builder - Backend

Backend REST API powering the Custom Drink Builder platform. Handles drink customization, pricing logic, menu management, and database operations.

## Frontend Repository

https://github.com/nishchay-raj/custom-drink-builder-frontend

## Features

- RESTful APIs
- Drink customization logic
- CRUD operations
- PostgreSQL database
- Prisma ORM
- Authentication (if implemented)

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Prisma ORM

## API Flow

Client->
Express Routes
->
Middleware
->
Controllers
->
Services
->
Prisma ORM
->
PostgreSQL

## Getting Started

```bash
pnpm install
pnpm run dev
```
